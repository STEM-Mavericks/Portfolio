Our Project

Automated Waste Segregation System

The Automated Waste Segregation System enhances environmental cleanliness by automating waste management. Our current basic model uses an Arduino Uno, servo motor, moisture sensor, and ultrasonic sensor. In the future, we plan to integrate additional features like a load cell, bin full alert, and web app development.

App Link

Check out our application: Visit our App WasteX

Key Features

  • Smart Waste Sorting: Automatically separates dry and wet waste with precision using moisture sensors.
  • Real-Time Monitoring: In the future, track bin levels and waste weight via a web application.
  • Overload Prevention: We plan to add an ultrasonic sensor to monitor bin capacity.
  • Sustainability Support: Our system will provide waste analytics, promoting environmental sustainability.
  • Durability: Designed to function effectively across various environmental conditions.

System Components

  • Sensors:
    • Moisture Sensor: Differentiates dry from wet waste.
    • Ultrasonic Sensor: Planned addition to track bin capacity and prevent overflow.
  • Actuators:
    • Servo Motor: Moves waste based on moisture levels.
    • Load Cell (Future): Will be used to measure waste weight.

How It Works

  • Moisture Detection: The sensor detects the moisture content of waste.
  • Sorting: The servo motor segregates waste into the appropriate compartment based on moisture levels.
  • Future Upgrades: Bin capacity monitoring and weight measurement will be integrated in later versions.

Environmental Impact

  • Reduces human error in waste sorting and encourages recycling.
  • Future versions will prevent bin overflow and provide detailed waste analytics.
  • Contributes to sustainable waste management practices.
