Our Project

Automated Waste Segregation System

The Automated Waste Segregation System enhances environmental cleanliness by automating waste management. This system is designed to make waste sorting easy, efficient, and reliable, helping users contribute to a cleaner world.

App Link

Check out our application: Visit our App WasteX

Key Features

  • Smart Waste Sorting: Automatically separates dry and wet waste with precision using moisture sensors.
  • Real-Time Monitoring: Keep track of bin levels and waste weight with real-time data, accessible via a web application.
  • Overload Prevention: The system monitors bin capacity with an ultrasonic sensor to avoid overflows.
  • Sustainability Support: Encourages efficient recycling by providing detailed waste analytics, promoting environmental sustainability.
  • Durability: Built for various environments—from humid to arid conditions—ensuring longevity and reliability.

System Components

  • Sensors:
    • Moisture Sensor: Differentiates dry from wet waste.
    • Load Sensor: Measures waste weight for tracking.
    • Ultrasonic Sensor: Tracks bin capacity, preventing overflow.
  • Actuators:
    • Motorized Actuator: Facilitates waste segregation by moving waste to the right compartment.
    • Conveyor/Sliding Mechanism: Ensures smooth movement of waste within the bin.

How It Works

  • Moisture Detection: Sensors detect the moisture content of the waste.
  • Sorting Mechanism: Depending on the moisture levels, the actuator sorts waste into the appropriate compartment.
  • Monitoring: Users can view real-time updates on waste levels and other analytics through the web interface.
  • Alerts: Notifications for bin fullness and other maintenance needs.
